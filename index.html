The Challenge

Find the maximum product between two numbers from the array, that is a multiple of 3

Given an array of integers, find the maximum product between two numbers from the array, that is a multiple of 3. The result for the input {6,8,8,7,2,5} should be 48 = 6*8. Note that 8*8 is the greatest product (64) but 64 isn't divisible by 3. Given the input {1,9,2,4} the result should be 36 = 9*4, and not 81 (9*9) since you should't multiply a number by itself.

Array of integer is required.
Input array of integers like {6,8,8,7,2,5}.

Invalid format or none values of Input array.
